* Level&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;   :  Easy
* Category :  Arrays<br><br>
*example 1*
```sh
   Input : 2 7 11 15
   target : 9
   Output : [0, 1]
   ```
*Note : because nums[0] + nums[1] == 9, we return [0, 1].*<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*example 2*
```sh
   Input : 3 2 4
   target : 6
   Output : [1, 2]
   ```
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*example 3*
```sh
   Input : 3 3
   target : 6
   Output : [0, 1]
   ```
